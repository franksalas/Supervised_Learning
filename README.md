# Supervised_Learning

## Enviroment setup

Create an enviroment from the `enviroment.yml` file
```bash
conda env create -f environment.yml
```

activate enviroment

```bash
source activate supervised
```

Update enviroment

```bash
conda env export > environment.yml
```
> [source](https://conda.io/docs/user-guide/tasks/manage-environments.html#creating-an-environment-from-an-environment-yml-file)
